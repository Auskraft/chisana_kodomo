# Звуки зверей для «Звуки»/«Ферма» — где брать (CC0 / без атрибуции)

Нужно: короткий звук зверя `assets/animals/<key>.wav` (моно, 44100 Гц, ~0.5–1.5 с).
Скачивай в любом формате/длине — **приведу к нужному скриптом** (`tool/prep_animal_sounds.py`,
кину по запросу): конвертация в WAV, обрезка тишины, нормализация громкости, имя по ключу.

## Источники (по удобству)

1. **Mixkit — animals** — без логина, сразу WAV, без атрибуции, можно в коммерции
   (Mixkit License: бесплатно для личного и коммерческого, нельзя перепродавать
   звук как отдельный товар). Есть: dog, cat, cow, horse, chicken, rooster, bird,
   wolf, lion, monkey, donkey. → https://mixkit.co/free-sound-effects/animals/
2. **Pixabay — Sound Effects** — огромный выбор по каждому зверю, без атрибуции,
   коммерция ок (Pixabay Content License). Нужен **бесплатный аккаунт** для скачивания.
   Поиск по зверю — прямые ссылки в таблице ниже. → https://pixabay.com/sound-effects/
3. **Freesound (фильтр CC0)** — для редких (змея, мышь, тюлень, дельфин…). Нужен
   бесплатный аккаунт; обязательно выбери лицензию **«Creative Commons 0»** в фильтре
   слева (иначе попадётся CC-BY с атрибуцией). → https://freesound.org/search/?f=license:%22Creative+Commons+0%22
4. **OpenGameArt — CC0-паки** (скачать пачкой): «Baby Animals — Sounds Pack» (CC0),
   «CC0 Sounds Library», «Animal Sounds». Проверяй лицензию у каждого пака (бывает CC-BY).
   → https://opengameart.org/content/baby-animals-sounds-pack ·
   https://opengameart.org/content/cc0-sounds-library

> Совет: сначала пройди **Mixkit** (быстро, без логина), что не нашлось — добери на
> **Pixabay**, совсем редкое — на **Freesound CC0**.

## Без иконичного звука — НЕ ищи, оставим голос

giraffe · zebra · hippo · rhino · panda · koala · hedgehog · squirrel · raccoon ·
kangaroo · deer · walrus · flamingo · swan · turtle · crab · ladybug · butterfly ·
snail · camel · sloth — у них нет узнаваемого для малыша звука. В игре они просто
**называются голосом** (TTS/пак) — это уже работает, ошибок нет.

## Формат и подключение

- Клади как `assets/animals/<key>.wav` (моно, 44100, ~1 с). Имена ключей — латиницей,
  как в таблице (`bear.wav`, `polar_bear.wav`…).
- Скачал в MP3/длинное/стерео? Не парься — **скрипт приведёт** (конверт + обрезка +
  нормализация). Скажи «давай скрипт» — кину `tool/prep_animal_sounds.py`.
- Атрибуции (даже где не обязательно — хороший тон) веду в `docs/ATTRIBUTIONS.md`.

> (опц.) Могу перевести звуки зверей на **MP3** вместо WAV — APK заметно легче
> (звук в 5–10× меньше). Тогда поменяю путь в коде на `.mp3`. Скажи, если надо.
