## 1. Ассеты — отдать агенту (он подключит в код)
- [ ] 🎨 **Раскраска — контурные PNG по темам и уровням** → клади в
      `assets/coloring/<тема>/<уровень>/` (темы: `animals`/`cars`/`nature`/`food`,
      уровни `1`..`5`; имена файлов любые, подхватятся сами). **Движок готов**
      (заливка / выбор темы **лентой-чипами** с эмодзи / уровни 1–5 / отмена-возврат /
      колор-пикер). Критерии: тёмный **замкнутый** контур, белый фон, ~600–1500 px.
      Сейчас `animals` (48), `cars` (42), `food` (30) наполнены, уровни 1–5;
      остаётся `nature`. Тема показывается в игре, только когда
      в ней есть картинки; уровень — когда уровней с картинками ≥2. Новая тема =
      новая папка + строка в `pubspec.yaml` (или скажи мне). Проверь на устройстве —
      подстрою порог заливки.
- [x] 🗜️ **Скрипты раскрасок** — `tool/import_coloring.py` (раскладывает мастер-папку
      `<src>/1..5/*.png` в `assets/coloring/<тема>/`, транслитерируя кириллицу→латиницу)
      и `tool/compress_coloring.py` (Pillow: до 1024 px + квантование палитры).
      Применены к `animals` (51→**18 МБ**), `cars` (38→**14.5 МБ**), `food` (30→**11 МБ**).
      После добавления новых PNG прогоняй: `python tool/compress_coloring.py`
      (агрессивнее — `--max 900 --colors 32`). **Оригиналы держи в мастер-папке** —
      скрипт пишет поверх файлов в `assets/`.
- [ ] 🔊 **Звуки зверей — CC0 `.wav`** (моно, 44100 Гц) в `assets/animals/` по
      ключам: `dog cat cow pig hen frog sheep horse chick rabbit` (→ `dog.wav` …).
      **Нужны для «Фермы»** (квиз «угадай звук»): без файлов загадка озвучивается
      именем (фактически = квиз по имени), с файлами — настоящий «угадай звук».
- [x] 🖼️ Арт-иконки зверей (10) — **переданы, подключены** (Ферма + квиз).
- [x] 🎮 Иконки игр (8) — **переданы, подключены**.

## Решения (зафиксировано)
- Раскраска — заливка тапом (flood fill) по растру; PNG (лучше) или JPG, не EPS/SVG.
  Организация — **тема × уровень**: `assets/coloring/<тема>/<уровень>/` (селектор тем —
  лента-чипы с эмодзи). Источник — Magnific Free (brgfx), подпись обязательна.
- «Ферма» = квиз «угадай звук» (со звёздами): играет звук зверя → выбери зверя.
  Загадка-звук = CC0 `.wav`; пока файла нет — имя голосом.
- «Уточка» заменена на «цыплёнка» (под имеющийся арт) — обратимо.
