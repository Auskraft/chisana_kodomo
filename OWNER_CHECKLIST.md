# OWNER_CHECKLIST — что нужно от владельца

> Живой список, чтобы ничего не потерять. Агент пишет код и проверяет
> `analyze`/`test`; **собрать APK, проверить «фил» на устройстве, запушить и
> принести контент — на владельце.** Отмечай `[x]` сделанное.

## 1. Ассеты — отдать агенту (он подключит в код)
- [ ] 🎨 **Раскраска — ~50 PNG-контуров** (сначала **5–10 на пробу**). Лицензия:
      **CC0** или **Magnific** (с подписью). Критерии: тёмный **замкнутый** контур,
      белый/прозрачный фон, **≥800 px**, **1 предмет** на картинку, без штриховки.
      Положить — скажу куда (заведу `assets/coloring/`). → агент соберёт заливку
      (flood fill).
- [ ] 🔊 **Звуки зверей — CC0 `.wav`** (моно, 44100 Гц) в `assets/animals/` по
      ключам: `dog cat cow pig hen frog sheep horse chick rabbit` (→ `dog.wav` …).
      **Нужны для «Фермы»** (квиз «угадай звук»): без файлов загадка озвучивается
      именем (фактически = квиз по имени), с файлами — настоящий «угадай звук».
- [x] 🖼️ Арт-иконки зверей (10) — **переданы, подключены** (Ферма + квиз).
- [x] 🎮 Иконки игр (8) — **переданы, подключены**.
- [ ] (опц.) `duck.png` + при желании вернуть «уточку» (сейчас вместо неё «цыплёнок»).

## 2. Контент — сгенерить самому (инструменты уже в `tool/`)
- [ ] 🗣️ **Голос-пак** (офлайн, приятнее системного TTS):
      `tool/gen_voice_samples.py` → выбрать голос → `tool/gen_voice_pack.py`
      (Silero). Пак опционален — TTS уже работает. Для девочки/нейтрали финальная
      фраза набора пока через TTS (клипы можно добавить позже).
- [ ] (опц.) Перегенерить звук при желании другого тембра:
      `tool/gen_sfx.py`, `tool/gen_notes.py`.

## 3. Проверка на устройстве («фил») — агент НЕ запускает
- [ ] Все **8 игр**: анимации, голос, «сок» (частицы/тряска), хит-тест.
- [ ] **Раскраска**: рисование пальцем, заливка областей, «по номерам».
- [ ] **Звук**: SFX, ноты ксилофона; (после файлов) реальные звуки зверей.
- [ ] **Обращение** мальчик/девочка/нейтрально (1-й запуск + Настройки).
- [ ] Иконки зверей в сетке «Фермы» (10) и в плитках квиза.

## Решения (зафиксировано)
- Раскраска — режим **А** (PNG-контур + flood fill), формат PNG (не EPS/SVG).
- «Ферма» = квиз «угадай звук» (со звёздами): играет звук зверя → выбери зверя.
  Загадка-звук = CC0 `.wav`; пока файла нет — имя голосом.
- «Уточка» заменена на «цыплёнка» (под имеющийся арт) — обратимо.
