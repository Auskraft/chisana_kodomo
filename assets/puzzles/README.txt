Картинки для игры «Пазлы» (необязательно).

Положи сюда КВАДРАТНЫЕ PNG/JPG (≈600–1024 px) — игра будет резать их на кусочки
вместо эмодзи. Имена файлов любые, подхватятся сами. Если папка пуста (только этот
файл) — используются эмодзи-запас.

Советы: яркие простые картинки без мелких деталей (детям 3–6 лет); один объект по
центру читается лучше всего. Прозрачный фон можно — под ним будет кремовый фон доски.
