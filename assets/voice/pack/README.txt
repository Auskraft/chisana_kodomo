Сюда кладутся клипы встроенного («офлайн») голоса.

Файлы: <key>.wav (один на фразу). Их генерит tool/gen_voice_pack.py выбранным
голосом (Silero / edge-tts). Полный список ключей и фраз — там же.

Пока клипов нет, приложение озвучивает фразы системным TTS (фолбэк). Как только
файлы появятся и в Настройках выбран «Встроенный голос» — играют эти клипы
(офлайн, без Google, одинаково у всех).

Этот README — просто заглушка, чтобы папка попадала в сборку.
